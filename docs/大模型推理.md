## 模型简介

PCL-LLM-Model 是鹏城自研的万亿参数大模型, 该模型在架构设计上借鉴了 DeepSeek-V3 和 Qwen3, 并具有昇腾亲和的特性。

| 属性         | 值                  |
| ------------ | ------------------- |
| **架构**     | MoE + GQA           |
| **路由专家** | 128                 |
| 共享专家     | 1                   |
| 专家分组     | 8                   |
| TOP K        | 2                   |
| **隐藏维度** | 7168                |
| FFN 维度     | 18432               |
| 专家维度     | 12288               |
| **网络层数** | 32                  |
| DensLayer    | 2                   |
| **GQA**      | attention heads=64, |
| 词表大小     | 163840              |


目前已在 2048卡（4096Die）的千卡集群上成功完成了万亿参数级 MoE 大模型的预训练任务， 跑完了 5T 数据集，训练时长为 30天。该模型在集群架构层面，采用 **16 节点 × 8 卡** 的超节点基础单元拓扑，节点内 8 张 910C 通过高带宽互联。
模型采用 **TP=2（张量并行）、PP=8（流水线并行）、EP=64（专家并行）** 的混合并行策略，在 700+ 节点规模上实现了高效分布式训练。

| 配置项    | 训练配置 | 说明                                |
| ------ | ---- | --------------------------------- |
| **TP** | 2    | Tensor Parallel（张量并行）             |
| **PP** | 8    | Pipeline Parallel（流水线并行）          |
| **EP** | 64   | Expert Parallel（专家并行）             |
| **CP** | 1    | Context Parallel（序列并行，可选 Ulysses） |
